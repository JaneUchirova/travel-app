# Travel App

A responsive React application for discovering trips, filtering destinations,
booking trips, and managing bookings.

## Live demo

[Open Travel App](https://effortless-nasturtium-1dff92.netlify.app/)

## Technologies

- React
- React Router
- Vite
- CSS

## Run locally

```bash
npm install
npm run dev
```

## Quality checks

```bash
npm run lint
npm run build
```
